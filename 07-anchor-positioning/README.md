# 07 — Anchor Positioning

## 🧠 Concept

This module explores **CSS Anchor Positioning**, a modern layout system that allows elements to be positioned relative to other elements using **semantic relationships instead of coordinates**.

Instead of:

- ❌ `position: absolute`
- ❌ manual `top / left calculations`
- ❌ JavaScript positioning logic

We use:

- ✅ `anchor-name`
- ✅ `position-anchor`
- ✅ `anchor()` functions

---

## 🎯 Goal

Build a UI system where elements:

- attach to other elements
- follow their position automatically
- adapt without JS or manual math

---

## 🧩 Demo Structure

This lab includes 3 anchor-based UI patterns:

### 1. Button → Tooltip

A tooltip is automatically positioned under a button.

### 2. Input → Helper Hint

A hint label follows the input field dynamically.

### 3. Card → Floating Actions

A contextual action bar is anchored to a card.

---

## 🔥 Key Idea

Anchor Positioning replaces coordinate-based UI with **relationship-based UI**.

> UI elements are no longer placed — they are connected.
