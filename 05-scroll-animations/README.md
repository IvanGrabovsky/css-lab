# Scroll Animations

## 🧠 What is it?

Scroll-driven Animations allow CSS animations to be controlled by the scroll position.

Instead of running over time, the animation progresses as the user scrolls.

---

## 🧩 Syntax

```css
animation-timeline: scroll();
```

---

## 💡 Example

An element fades in and moves upward while the page scrolls.

---

## 📖 Explanation

Traditional animations are based on time.

Scroll animations are based on scroll progress.

This enables smooth, interactive experiences without JavaScript.

---

## 🚀 Use Cases

- Reading progress indicators
- Reveal-on-scroll effects
- Interactive storytelling
- Product landing pages
- Portfolio websites
