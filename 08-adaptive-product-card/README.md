# Adaptive Product Card

## Goal

Build a real product card by combining multiple modern CSS features.

---

## Features

- clamp()
- :has()
- Container Queries

---

## What this demo shows

- Fluid spacing and typography
- Responsive layout without media queries
- Component-level responsiveness
- Pure CSS state management

---

## Key takeaway

Modern CSS features become much more powerful when combined.

Instead of solving isolated problems, they allow components to become responsive, interactive and reusable without JavaScript.

---

## Technologies

- HTML5
- CSS Grid
- clamp()
- :has()
- Container Queries
