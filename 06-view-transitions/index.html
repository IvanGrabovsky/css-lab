View Transitions

1 — JavaScript API

Use document.startViewTransition() to wrap any DOM mutation. The browser captures a before/after snapshot and plays the CSS transition between them.

2 — CSS-native Navigation

With @view-transition { navigation: auto; } in CSS, browser navigations between same-origin pages are automatically animated — no JavaScript needed.
