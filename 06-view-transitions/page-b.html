No JavaScript was used for this page transition. The browser applied the cross-fade animation automatically because both pages declare:

How it works

When the browser navigates between two same-origin pages that both opt-in via @view-transition, it captures screenshots of both pages and cross-fades them using the ::view-transition-* pseudo-elements.
